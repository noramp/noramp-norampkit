# Quick start

```bash

yarn add @noramp/norampkit

```

```javascript
import { NoRampPayButton } from '@noramp/norampkit';

<NoRampPayButton
  apiKey="API_KEY"
  appId="APP_ID"
  triggerId="TRIGGER_ID"
  currency="usd"
  triggerParams={['MINTER_ADDRESS']}
  priceInFiat={5}
  production={true}
/>;
```

### Running examples

Go into an example directory, eg:

```bash
cd examples/
```

Install dependencies.

```bash
yarn
```

Then run the dev script.

```bash
yarn start
```
